# E2E Release Evidence Assets

This directory is reserved for sanitized, intentionally tracked screenshots or other small visual assets from release rehearsals.

Raw captures belong in the gitignored `artifacts/e2e-release-rehearsal/` directory. Promote an asset here only when it clarifies stable public behavior, contains no private local state, and is safe to reference from README or release docs with a repo-relative link.
